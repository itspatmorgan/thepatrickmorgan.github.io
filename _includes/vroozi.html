I moved to Los Angeles in June 2016 to start a new adventure after spending a few years in New York. I linked up with LA native fin-tech startup Vroozi (based in Sherman Oaks) to help continue their product development and in particular to drive their push toward 100% mobile enabled business purchasing and expense management.

Responsive UI + Emails

Vroozi is focused on delivering a mobile friendly procurement platform, but it wasn't initially built that way. I'm working on transitioning the product to a truly mobile first approach for design and development.

UI Code Guidelines

Vroozi uses a lot of contract developers all over the world but doesn't have a clear code style guide for those developers to follow. I'm helping to create those guidelines while also reviewing and managing all the UI related pull requests going into the platform.

Component Architecture

Vroozi is built with Angular and we have been gradually adopting the ES6 component pattern within the framework for new UI features. I'm particularly interested in figuring out the best way to also develop modular styles to better support the component based system.
