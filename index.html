American Express

Front End Development + UX Design

Over the last year, I put extra focus on researching front-end best practices and helping them find their way into our legacy code base. Tools like Grunt, SCSS-lint and CSScomb helped us automate mundane tasks and spend more time building a better experience. Better class naming strategy and abstraction patterns helped us simplify our code for better reuse over time. The goal is to drastically simplify the front-end code which will let us prototype more efficiently, deliver a more consistently excellent user experience and maintain cleaner code in production. That means less bug fixing, fewer workarounds, and more productivity all around – never a bad thing.
