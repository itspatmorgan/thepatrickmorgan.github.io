Redesigning the website using responsive design was a huge undertaking involving pretty much every team in our organization at American Express. However once the redesign was in place, overall development and design time decreased significantly. We consolidated the code base. So even though it was still a large and complex code base, at least we didn't have to maintain two entirely separate sets or markup, styles and scripts for a mobile site.

While we embraced that ability to create a responsive design, we didn't get it perfect on day one. Day one was very much and MVP of responsive implementation that need to evolve, get leaner and cleaner over time.
